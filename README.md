# Computer 4 Kids

An interactive computer learning platform for children ages 4-12. Learn mouse skills, keyboard typing, internet safety, and more through fun, engaging lessons.

## Features

- Interactive lessons for mouse, keyboard, typing, and computer basics
- Progress tracking and gamification
- Multiple child profiles per account
- Daily rewards and streak tracking
- Safe, kid-friendly interface
- Parent dashboard

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Database & Authentication)

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:5173
